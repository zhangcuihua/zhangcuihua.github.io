==与===区别

=== 全等：判断值和类型是否相同

== 相等：只判断值是否相等

引用和值类型比较，转换规则：将引用类型转成值类型在进行比较。

转换方式

  • 会先调用引用类型的valueOf方法，如果这个值是值类型，就直接参与比较。
  • 如果valueof获取的值，不是值类型，会继续调用toString();把引用类型转成值类型的字符串，再进行比较。

==两边类型必须一致才能进行比较，如果两边都是值类型，遵循以下转换规则。

+ 默认null == undefined 结果为true。
  > console.log(null == undefined);   //true
+ 如果在==两边有true或者false先把true转成数字1，把false转成0进行比较。
+ 如果是字符串和数字进行比较，把字符串转成数字进行比较。
